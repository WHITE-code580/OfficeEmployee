“OfficeEmployee”.

A simple JavaScript object-oriented programming (OOP) project that models office employees and clerical officers. 
The application demonstrates core OOP concepts such as inheritance, encapsulation, constructors, getters, 
instance methods, and static methods.

Features:

•	Create office employee records with randomly generated names. 

•	Track the number of days remaining before transitioning to a higher position. 

•	Create clerical officers with assigned departments. 

•	Add new departments to existing clerical officers. 

•	Demonstrate inheritance through a parent-child class structure. 

Technologies:

•	JavaScript  

Concepts Demonstrated:

•	Classes and Objects 

•	Inheritance (extends) 

•	Constructors 

•	Getters 

•	Instance Methods 

•	Static Methods 

•	Array Manipulation 

Example Output:

Clerical Officer: Jane

Transitioning to Higher Position Days: 20

Departments: HR, Finance

Purpose:

This project was created as a learning exercise to practice object-oriented programming principles in JavaScript 
through a simple office employee management scenario.
